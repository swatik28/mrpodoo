Stock Aging Report for Odoo 18

This module provides a Stock Aging Report It allows users to export a Excel sheet with detailed information on quantity and value per aging period.

Key Features:

  • Export Excel report with inventory aging buckets
  • Filter by Product or Product Category
  • Select Warehouses and Internal Locations
  • Select Company and Date filters
  • Automatic calculation of Quantity and Value per aging range
  • Clean and readable Excel formatting

How to Use:

  1. Go to Inventory > Reporting > Stock Aging Report
  2. Fill in the filters: Date, Warehouses, Products , etc.
  3. Click onPrint Excel to download the excel report
